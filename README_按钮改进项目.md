**代码示例：**

```python
# 创建 Primary 按钮（主操作）
execute_btn = ModernButton(root, text="▶ 执行", 
                          button_type="primary",
                          command=on_execute)
execute_btn.pack(fill="x")

# 创建 Secondary 按钮（次操作）
import_btn = ModernButton(root, text="📂 导入", 
                         button_type="secondary",
                         command=on_import)
import_btn.pack(side="left", padx=4)

# 初始禁用
execute_btn.set_disabled(True)

# 条件启用
if ready:
    execute_btn.set_disabled(False)
```

## 🎯 核心要点速记

### 三种按钮
```
Primary → 橙色 (#FF6B35) → 主操作 → 一页面最多 1 个
Secondary → 蓝灰 (#2A3F6B) → 次操作 → 可多个
Danger → 红色 (#EE5A6F) → 危险 → 待扩展
```

### 四种状态
```
默认 → 基础颜色
Hover → 变亮 + relief="raised"（浮起）
Active → 变深 + relief="sunken"（下压）
Disabled → 灰色 + 不可点
```

### 一个方法
```python
btn.set_disabled(True/False)  # 启用/禁用
```
