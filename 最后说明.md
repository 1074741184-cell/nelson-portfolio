## 💡 核心功能一览

### 三种按钮类型

**Primary（主操作）** - 橙色
```python
ModernButton(root, text="▶ 执行", button_type="primary")
```
- 颜色：#FF6B35（橙色）
- 用途：一页面最多 1 个主按钮
- 示例：执行、提交、保存

**Secondary（次操作）** - 蓝灰色
```python
ModernButton(root, text="导入", button_type="secondary")
```
- 颜色：#2A3F6B（深蓝灰）
- 用途：辅助功能，可多个
- 示例：导入、删除、预览

**Danger（危险操作）** - 红色
```python
ModernButton(root, text="删除", button_type="danger")
```
- 颜色：#EE5A6F（红色）
- 用途：破坏性操作
- 示例：删除、重置

### 完整交互系统

```
🖱️ Hover   → 背景变亮 + 边框浮起
🖱️ Active  → 背景变深 + 边框下压
🖱️ Release → 保持亮色 + 执行操作
🖱️ Disabled → 变灰 + 禁用
```

### 简单的状态管理

```python
btn.set_disabled(True)   # 禁用
btn.set_disabled(False)  # 启用
```
