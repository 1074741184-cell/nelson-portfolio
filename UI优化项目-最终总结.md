```python

self.card_bg = "#1A1F3A"         # 卡片背景
self.border_color = "#2A3F6B"    # 边框色

# 强调色
self.theme_orange = "#FF6B35"    # 橙红强调
self.theme_green = "#00D084"     # 绿色成功
self.theme_gold = "#FFB81C"      # 金色警告

# 文本色
self.text_light = "#E0E6FF"      # 浅蓝文本
self.text_muted = "#7A8BA8"      # 灰蓝文本
```

```

输入框      #253355  → 交互区域
边框        #2A3F6B  → 框线
─────────────────────────────
青色        #00E5FF  → 重要标题
```

## 💡 设计思路

### 为什么这样改？

```
【深蓝背景】
    ↓
更高级、更科技、不显疲劳

【彩色强调】
    ↓
信息区分清晰、视觉丰富

【边框和hover】
    ↓
交互反馈强、用户清楚能干什么

【emoji和符号】
    ↓
视觉生动、一眼识别功能

【统一灰色】
    ↓
减少颜色混乱、提高专业度
```

### 设计原则

```
1. 对比原则：主要和辅助色要有明确对比
2. 一致性：同类元素样式保持一致
3. 层级性：用颜色/大小/位置表现优先级
4. 反馈性：用户操作要有立即反馈
5. 简洁性：不过度装饰，保持简洁
6. 可用性：考虑用户的易用性
```
