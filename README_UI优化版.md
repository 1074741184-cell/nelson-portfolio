## 🎯 设计原则

### 遵循的设计思路

```
1. 对比原则
   - 主次色要有明确对比
   - 提高可读性

2. 一致性原则
   - 同类元素样式一致
   - 构建统一设计系统

3. 层级原则
   - 用颜色/大小/位置表现优先级
   - 引导用户注意力

4. 反馈原则
   - 所有交互都有反馈
   - 提高交互感

5. 简洁原则
   - 不过度装饰
   - 保持简洁专业
```
